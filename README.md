# @systemlight/webpack5-react

> Front end development template based on webpack5.

## Issue

macOS下husky不执行？

> 设置权限使用 `chmod 777 .husky/*`

---

windows环境powershell下pnpm不执行问题？

> 管理员模式打开powershell运行如下命令

```shell
Set-ExecutionPolicy RemoteSigned
```

如何局部忽略prettier格式化？
